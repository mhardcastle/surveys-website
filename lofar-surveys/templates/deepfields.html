Deep fields

This page provides KSP access to the radio and multiwavelength data for the three Northern-sky deep fields (Boötes, Lockman Hole and ELAIS-N1).

For the radio data, all images have been flux-corrected and PyBDSF catalogues have been generated. For general science we recommend you use the radio images in the centre of the field (full-resolution, blanked at 0.3 PB, cropped) but other images are available if needed. Stokes QU cubes can be obtained by contacting Martin or Tim.

For optical data and crossmatches between radio and optical, the 'catalogue page' links for each field point to the multi-wavelength data in each field, and the source association and cross-match identifications. Please see the README documents on those pages for more information about what is provided.
